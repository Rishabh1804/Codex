# AI in Finance

> Algorithmic trading, risk modeling, fraud detection, and machine learning for financial applications.

### Courses

- [NYU: Machine Learning in Finance](https://www.coursera.org/learn/machine-learning-in-finance) - ML applications in finance from NYU Tandon. `Intermediate`
- [Coursera: Machine Learning for Trading Specialization](https://www.coursera.org/specializations/machine-learning-trading) - Google Cloud specialization on trading algorithms and ML. `Intermediate`
- [QuantInsti: Algorithmic Trading Courses](https://quantra.quantinsti.com/courses) - Free courses on quantitative trading strategies. `Intermediate`
- [Udacity: AI for Trading](https://www.udacity.com/course/ai-for-trading--nd880) - Free preview content on AI-powered trading strategies. `Intermediate`
- [WorldQuant University: Data Science Module](https://www.wqu.edu/) - Free applied data science courses with financial focus. `Beginner`

### Books & Reading

- [Advances in Financial ML (de Prado)](https://www.wiley.com/en-us/Advances+in+Financial+Machine+Learning-p-9781119482086) - Resources and code for the influential financial ML book. `Advanced`
- [Python for Finance (Yves Hilpisch)](https://home.tpq.io/books/py4fi/) - Resources and notebooks for financial Python programming. `Intermediate`
- [Machine Learning for Algorithmic Trading (GitHub)](https://github.com/stefan-jansen/machine-learning-for-trading) - Free code and notebooks from the popular ML for trading book. `Intermediate`

### Tools & Practice

- [Kaggle: Financial Datasets](https://www.kaggle.com/datasets?tags=13302-Finance) - Real-world financial datasets for practice. `All Levels`
- [QuantLib (Documentation)](https://www.quantlib.org/) - Open-source library for quantitative finance. `Advanced`
- [Zipline](https://github.com/quantopian/zipline) - Pythonic algorithmic trading library for backtesting. `Intermediate`
- [Microsoft Forecasting Repository](https://github.com/microsoft/forecasting) - Financial forecasting tutorials and best practices. `Intermediate`
- [SSRN: Financial ML Papers](https://papers.ssrn.com/sol3/Jeljour_results.cfm?form_name=journalBrowse&journal_id=3930309) - Free access to financial ML research papers. `Advanced`
