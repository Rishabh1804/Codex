# AI Security & Privacy

> Adversarial ML, federated learning, differential privacy, and protecting AI systems and user data.

### Courses

- [OpenMined: Privacy-Preserving AI](https://courses.openmined.org/) - Free courses on federated learning and differential privacy. `Intermediate`
- [Coursera: Privacy in the Digital Age](https://www.coursera.org/learn/privacy-in-the-digital-age) - Privacy-preserving techniques and their applications. `Beginner`
- [Adversarial Machine Learning Tutorial](https://adversarial-ml-tutorial.org/) - Comprehensive guide to adversarial attacks and defenses. `Intermediate`
- [NVIDIA: AI Security Fundamentals](https://www.nvidia.com/en-us/training/) - Security considerations in AI system design. `Intermediate`

### Books & Reading

- [The Algorithmic Foundations of Differential Privacy](https://www.cis.upenn.edu/~aaroth/Papers/privacybook.pdf) - Definitive free textbook on differential privacy theory. `Advanced`
- [Differential Privacy Resources](https://differentialprivacy.org/resources/) - Collection of papers, tutorials, and code on differential privacy. `Intermediate`
- [NIST AI Risk Management Framework](https://airc.nist.gov/AI_RMF_Interactivity/Playbook) - US government framework for managing AI risks. `Intermediate`
- [OWASP ML Security Top 10](https://owasp.org/www-project-machine-learning-security-top-10/) - Top security risks in machine learning. `Intermediate`

### Tools & Frameworks

- [Google: Federated Learning](https://federated.withgoogle.com/) - Resources and tutorials on privacy-preserving ML. `Intermediate`
- [PySyft (OpenMined)](https://github.com/OpenMined/PySyft) - Library for secure and private computation with tutorials. `Intermediate`
- [Microsoft Counterfit](https://github.com/Azure/counterfit) - Tool for assessing ML model security. `Advanced`
- [Adversarial Robustness Toolbox (IBM)](https://adversarial-robustness-toolbox.readthedocs.io/) - Library for defending against adversarial attacks. `Advanced`
- [TensorFlow Privacy](https://github.com/tensorflow/privacy) - Library for training ML models with differential privacy. `Advanced`
- [Flower (Federated Learning)](https://flower.ai/) - Open-source federated learning framework with tutorials. `Intermediate`
