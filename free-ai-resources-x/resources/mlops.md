# MLOps & Model Deployment

> Production ML pipelines, model serving, monitoring, CI/CD for ML, and taking models from notebook to production.

### Courses

- [Coursera: MLOps Specialization (DeepLearning.AI)](https://www.coursera.org/specializations/machine-learning-engineering-for-production-mlops) - Andrew Ng's comprehensive MLOps specialization. `Intermediate`
- [Made With ML: MLOps Course](https://madewithml.com/) - Comprehensive free guide to building production ML systems. `Intermediate`
- [Full Stack Deep Learning](https://fullstackdeeplearning.com/) - From training models to deploying and monitoring in production. `Intermediate`
- [Google Cloud: MLOps Learning Path](https://www.cloudskillsboost.google/paths/17) - Free path covering ML pipeline automation and deployment. `Intermediate`
- [Microsoft: MLOps with Azure ML](https://learn.microsoft.com/en-us/training/paths/introduction-machine-learn-operations/) - Free learning path on MLOps with Azure. `Intermediate`

### Books & Guides

- [MLOps Guide by Chip Huyen](https://huyenchip.com/mlops/) - Comprehensive guide and curated resources on MLOps practices. `Intermediate`
- [Designing ML Systems (Chip Huyen) — Resources](https://github.com/chiphuyen/dmls-book) - Free companion resources for the popular ML systems design book. `Intermediate`
- [Reliable Machine Learning (O'Reilly)](https://www.oreilly.com/library/view/reliable-machine-learning/9781098106218/) - Resources on building reliable ML systems. `Advanced`

### Tools & Platforms

- [MLflow Documentation](https://mlflow.org/docs/latest/index.html) - Open-source platform for the complete ML lifecycle. `Intermediate`
- [BentoML Documentation](https://docs.bentoml.com/) - Framework for serving, managing, and deploying ML models. `Intermediate`
- [Kubeflow Documentation](https://www.kubeflow.org/docs/) - ML toolkit for Kubernetes with pipelines and model serving. `Advanced`
- [Seldon Core Documentation](https://docs.seldon.io/projects/seldon-core/en/latest/) - Open-source platform for deploying ML models on Kubernetes. `Advanced`
- [Great Expectations](https://docs.greatexpectations.io/) - Data quality and validation for ML pipelines. `Intermediate`
- [Evidently AI](https://docs.evidentlyai.com/) - Open-source ML monitoring and observability tool. `Intermediate`
