# Computer Vision

> Image processing, object detection, segmentation, visual recognition, and everything related to teaching machines to see.

### Courses

- [Stanford CS231n: Deep Learning for Computer Vision](https://cs231n.github.io/) - The gold standard CV course — CNNs, detection, segmentation, and more. `Intermediate`
- [Coursera: Introduction to Computer Vision (Georgia Tech)](https://www.coursera.org/learn/introduction-computer-vision) - Comprehensive course covering fundamental CV algorithms. `Beginner`
- [OpenCV University: Free Courses](https://opencv.org/university/free-courses/) - Hands-on courses on computer vision and image processing with OpenCV. `Beginner`
- [First Principles of Computer Vision (Columbia)](https://www.youtube.com/channel/UCf0WB91t8Ky6AuYcQV0CcLw) - Excellent YouTube lecture series covering CV from the ground up. `Beginner`
- [Michigan: Deep Learning for Computer Vision](https://web.eecs.umich.edu/~justliu/teaching/eecs498/) - Justin Johnson's CV course (creator of CS231n assignments). `Intermediate`
- [Kaggle: Computer Vision Course](https://www.kaggle.com/learn/computer-vision) - Free hands-on course for building CV models in the browser. `Beginner`

### Books & Reading

- [Computer Vision: Algorithms and Applications (Szeliski)](https://szeliski.org/Book/) - Comprehensive free textbook covering classical and modern CV. `Intermediate`
- [PyImageSearch Tutorials](https://pyimagesearch.com/start-here/) - Extensive practical tutorials on OpenCV, deep learning, and CV. `Beginner`
- [Programming Computer Vision with Python (Solem)](http://programmingcomputervision.com/) - Free book on implementing CV algorithms from scratch. `Intermediate`

### Tools & Practice

- [OpenCV Documentation & Tutorials](https://docs.opencv.org/4.x/d6/d00/tutorial_py_root.html) - Official Python tutorials for the most popular CV library. `All Levels`
- [Roboflow Universe](https://universe.roboflow.com/) - Free datasets and pre-trained models for CV projects. `All Levels`
- [Ultralytics YOLOv8 Docs](https://docs.ultralytics.com/) - State-of-the-art object detection with tutorials and guides. `Intermediate`
- [Segment Anything (Meta)](https://segment-anything.com/) - Foundation model for image segmentation with demos and code. `Intermediate`
- [Hugging Face: Computer Vision Models](https://huggingface.co/models?pipeline_tag=image-classification) - Thousands of pre-trained vision models with documentation. `All Levels`
