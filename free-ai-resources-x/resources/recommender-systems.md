# Recommender Systems

> Collaborative filtering, content-based filtering, hybrid methods, and personalization algorithms.

### Courses

- [Google: Recommendation Systems Course](https://developers.google.com/machine-learning/recommendation) - Comprehensive guide to building recommendation systems from Google. `Beginner`
- [Coursera: Recommender Systems Specialization (Minnesota)](https://www.coursera.org/specializations/recommender-systems) - In-depth specialization covering collaborative filtering, content-based, and hybrid methods. `Intermediate`
- [Stanford CS246: Mining Massive Datasets (RecSys)](https://web.stanford.edu/class/cs246/) - Recommendation systems within the context of large-scale data mining. `Intermediate`
- [fast.ai: Collaborative Filtering](https://course.fast.ai/) - Practical deep learning with collaborative filtering examples. `Beginner`

### Books & Reading

- [Recommender Systems Handbook (Ricci et al.)](https://link.springer.com/book/10.1007/978-1-0716-2197-4) - Comprehensive reference on recommender system techniques. `Advanced`
- [Recommender Systems: The Textbook (Aggarwal)](https://link.springer.com/book/10.1007/978-3-319-29659-3) - Broad coverage of recommendation algorithms and techniques. `Intermediate`
- [Netflix Prize Papers](https://www.netflixprize.com/) - Historical resources from the competition that advanced the field. `Intermediate`

### Libraries & Practice

- [Microsoft Recommenders (GitHub)](https://github.com/microsoft/recommenders) - Best practices and reference implementations for recommendation systems. `Intermediate`
- [Surprise Library](http://surpriselib.com/) - Python scikit for building and analyzing recommender systems. `Beginner`
- [LensKit](https://lenskit.org/) - Open-source toolkit for building and evaluating recommender systems. `Intermediate`
- [RecBole](https://recbole.io/) - Unified framework for developing and reproducing recommendation algorithms. `Intermediate`
- [ACM RecSys Resources](https://github.com/ACMRecSys) - Conference resources and educational materials. `All Levels`
- [Merlin (NVIDIA)](https://nvidia-merlin.github.io/Merlin/main/README.html) - GPU-accelerated framework for large-scale recommendation systems. `Advanced`
