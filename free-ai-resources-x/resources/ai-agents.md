# AI Agents & Agentic AI

> Autonomous AI systems, tool-using LLMs, multi-agent frameworks, and building AI that can take actions.

### Courses & Tutorials

- [DeepLearning.AI: AI Agents in LangGraph](https://www.deeplearning.ai/short-courses/ai-agents-in-langgraph/) - Building AI agents with LangGraph framework. `Intermediate`
- [DeepLearning.AI: Functions, Tools and Agents with LangChain](https://www.deeplearning.ai/short-courses/functions-tools-agents-langchain/) - Building tool-using AI agents. `Intermediate`
- [DeepLearning.AI: Multi AI Agent Systems with crewAI](https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/) - Designing multi-agent collaboration systems. `Intermediate`
- [Hugging Face: Building AI Agents](https://huggingface.co/learn/agents-course/unit0/introduction) - Free course on building agents with open-source tools. `Beginner`

### Frameworks & Tools

- [LangChain Documentation](https://python.langchain.com/docs/get_started/introduction) - Framework for building LLM-powered applications and agents. `Intermediate`
- [LangGraph Documentation](https://langchain-ai.github.io/langgraph/) - Framework for building stateful, multi-step AI agents. `Intermediate`
- [CrewAI Documentation](https://docs.crewai.com/) - Framework for orchestrating multi-agent AI systems. `Intermediate`
- [AutoGen (Microsoft)](https://microsoft.github.io/autogen/) - Framework for building multi-agent conversational systems. `Intermediate`
- [Semantic Kernel (Microsoft)](https://learn.microsoft.com/en-us/semantic-kernel/overview/) - SDK for integrating LLMs into applications with agent capabilities. `Intermediate`
- [Anthropic: Tool Use Documentation](https://docs.anthropic.com/en/docs/build-with-claude/tool-use) - Building tool-using AI agents with Claude. `Intermediate`
