# Machine Learning

> Foundational algorithms, techniques, and theory for building intelligent systems that learn from data.

### Courses

- [Coursera: Machine Learning by Andrew Ng](https://www.coursera.org/learn/machine-learning) - The gold standard introductory ML course covering supervised/unsupervised learning, best practices. `Beginner`
- [Google: Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) - Fast-paced, interactive course with TensorFlow exercises and real-world case studies. `Beginner`
- [fast.ai: Practical Deep Learning for Coders](https://course.fast.ai/) - Top-down, hands-on approach to learning ML and deep learning. `Beginner`
- [Kaggle: Intro to Machine Learning](https://www.kaggle.com/learn/intro-to-machine-learning) - Short, practical course with real datasets and in-browser coding. `Beginner`
- [Stanford CS229: Machine Learning](https://cs229.stanford.edu/) - Andrew Ng's full Stanford course with lecture notes, problem sets, and exams. `Intermediate`
- [Coursera: Applied Machine Learning in Python](https://www.coursera.org/learn/python-machine-learning) - Hands-on ML with scikit-learn covering classification, regression, and evaluation. `Intermediate`
- [Caltech: Learning from Data](https://home.work.caltech.edu/telecourse.html) - Rigorous ML theory course with video lectures and homework. `Intermediate`
- [Cornell CS4780: Machine Learning for Intelligent Systems](https://www.cs.cornell.edu/courses/cs4780/2018fa/) - Free lecture videos and notes on ML fundamentals. `Intermediate`

### Books & Reading

- [An Introduction to Statistical Learning (ISLR)](https://www.statlearning.com/) - Classic free textbook with R/Python labs covering regression, classification, resampling, and more. `Beginner`
- [Hands-On Machine Learning (GitHub Notebooks)](https://github.com/ageron/handson-ml3) - Free Jupyter notebooks companion to the popular Scikit-Learn and TensorFlow book. `Intermediate`
- [Pattern Recognition and Machine Learning (Bishop)](https://www.microsoft.com/en-us/research/publication/pattern-recognition-machine-learning/) - Classic ML textbook now freely available from Microsoft Research. `Advanced`

### Practice & Tools

- [Scikit-learn Documentation & Tutorials](https://scikit-learn.org/stable/tutorial/index.html) - Official tutorials for the most popular Python ML library. `Beginner`
- [Kaggle Competitions](https://www.kaggle.com/competitions) - Real-world ML competitions to practice and build your portfolio. `All Levels`
- [MLflow Documentation](https://mlflow.org/docs/latest/index.html) - Free open-source platform for managing the ML lifecycle. `Intermediate`
