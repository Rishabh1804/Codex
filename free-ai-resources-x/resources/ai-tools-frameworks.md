# AI Tools & Frameworks

> Essential libraries, frameworks, and platforms for building AI and ML systems.

### Deep Learning Frameworks

- [PyTorch: Official Tutorials](https://pytorch.org/tutorials/) - Step-by-step tutorials from basics to advanced — the most popular research framework. `All Levels`
- [TensorFlow: Official Tutorials](https://www.tensorflow.org/tutorials) - Comprehensive tutorials for Google's ML framework. `All Levels`
- [Keras: Getting Started](https://keras.io/getting_started/) - High-level neural network API with beginner-friendly tutorials. `Beginner`
- [JAX Tutorials (Google)](https://jax.readthedocs.io/en/latest/tutorials.html) - High-performance numerical computing and ML research framework. `Advanced`
- [fast.ai Library Documentation](https://docs.fast.ai/) - High-level PyTorch wrapper that simplifies training. `Beginner`

### ML Libraries & Tools

- [Scikit-learn Tutorials](https://scikit-learn.org/stable/tutorial/index.html) - The go-to Python library for classical ML algorithms. `Beginner`
- [Hugging Face Transformers](https://huggingface.co/docs/transformers/index) - Industry-standard library for NLP, vision, and audio models. `Intermediate`
- [LangChain Documentation](https://python.langchain.com/docs/get_started/introduction) - Framework for building LLM-powered applications. `Intermediate`
- [Weights & Biases (W&B) Documentation](https://docs.wandb.ai/) - Free experiment tracking, visualization, and collaboration platform. `Beginner`
- [Lightning AI Documentation](https://lightning.ai/docs/pytorch/stable/) - Framework for organizing PyTorch code and scaling training. `Intermediate`

### Data & Experiment Tools

- [Pandas Documentation](https://pandas.pydata.org/docs/getting_started/index.html) - Essential data manipulation library for Python. `Beginner`
- [DVC (Data Version Control)](https://dvc.org/doc) - Open-source tool for ML experiment management and data versioning. `Intermediate`
- [Label Studio](https://labelstud.io/) - Open-source data labeling tool for ML projects. `Beginner`
- [Optuna Documentation](https://optuna.readthedocs.io/) - Automatic hyperparameter optimization framework. `Intermediate`
- [ONNX (Open Neural Network Exchange)](https://onnx.ai/) - Open format for sharing ML models between frameworks. `Intermediate`
