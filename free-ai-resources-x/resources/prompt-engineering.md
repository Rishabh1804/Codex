# Prompt Engineering

> Techniques for crafting effective prompts, optimizing LLM outputs, and mastering human-AI interaction.

### Courses

- [DeepLearning.AI: ChatGPT Prompt Engineering for Developers](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/) - Hands-on course on prompt engineering with APIs. `Beginner`
- [DeepLearning.AI: Building Systems with ChatGPT](https://www.deeplearning.ai/short-courses/building-systems-with-chatgpt/) - Designing multi-step AI systems using prompts. `Intermediate`
- [Google: Prompt Design in Vertex AI](https://www.cloudskillsboost.google/course_templates/976) - Free course on prompt design for Google's AI platform. `Beginner`

### Guides & Documentation

- [Anthropic: Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview) - Comprehensive official guide to prompting Claude. `All Levels`
- [OpenAI: Prompt Engineering Guide](https://platform.openai.com/docs/guides/prompt-engineering) - Official best practices for prompting GPT models. `All Levels`
- [Prompt Engineering Guide (DAIR.AI)](https://www.promptingguide.ai/) - Community-driven comprehensive guide to prompt techniques. `All Levels`
- [Learn Prompting](https://learnprompting.org/) - Free open-source guide covering beginner to advanced prompting. `All Levels`
- [Google: Prompting Guide for Gemini](https://ai.google.dev/gemini-api/docs/prompting-intro) - Official prompting guide for Google's Gemini models. `Beginner`

### Advanced Techniques & Practice

- [PromptHero](https://prompthero.com/) - Community platform for sharing and discovering prompts. `Beginner`
