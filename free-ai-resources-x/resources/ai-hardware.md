# AI Hardware & Infrastructure

> GPUs, TPUs, distributed training, cloud computing, and the hardware powering modern AI.

### Courses & Tutorials

- [NVIDIA Deep Learning Institute](https://www.nvidia.com/en-us/training/) - Free courses on GPU computing, CUDA, and AI acceleration. `Intermediate`
- [Google Cloud: TPU Tutorials](https://cloud.google.com/tpu/docs/tutorials) - Guides on using TPUs for ML training. `Intermediate`
- [AWS: Deep Learning AMI Guide](https://aws.amazon.com/getting-started/hands-on/get-started-dlami/) - Free tier tutorials on cloud ML infrastructure. `Beginner`
- [Harvard CS249r: TinyML](https://www.edx.org/learn/computer-science/harvard-university-cs249r-tiny-machine-learning-tinyml) - ML for edge devices and resource-constrained hardware. `Intermediate`
- [CUDA Programming Guide (NVIDIA)](https://docs.nvidia.com/cuda/cuda-c-programming-guide/) - Official guide to GPU programming with CUDA. `Advanced`

### Distributed Training

- [Distributed Training with Horovod](https://horovod.readthedocs.io/en/stable/) - Documentation on distributed deep learning across GPUs. `Advanced`
- [Ray Documentation](https://docs.ray.io/en/latest/ray-overview/getting-started.html) - Tutorials on scaling ML workloads across clusters. `Intermediate`
- [DeepSpeed (Microsoft)](https://www.deepspeed.ai/) - Deep learning optimization library for distributed training. `Advanced`
- [PyTorch Distributed Tutorial](https://pytorch.org/tutorials/beginner/dist_overview.html) - Official guide to distributed training in PyTorch. `Intermediate`
- [FSDP (Fully Sharded Data Parallel)](https://pytorch.org/tutorials/intermediate/FSDP_tutorial.html) - PyTorch's native large-model training approach. `Advanced`
