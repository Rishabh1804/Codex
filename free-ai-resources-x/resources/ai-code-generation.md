# AI for Code Generation

> AI-assisted programming, code completion, automated testing, and tools that help developers write better code.

### Courses

- [DeepLearning.AI: Pair Programming with LLMs](https://www.deeplearning.ai/short-courses/pair-programming-llm/) - Course on AI-assisted programming techniques. `Beginner`
- [DeepLearning.AI: AI Python for Beginners](https://www.deeplearning.ai/short-courses/ai-python-for-beginners/) - Learn Python with AI assistance. `Beginner`
- [Microsoft: Introduction to GitHub Copilot](https://learn.microsoft.com/en-us/training/modules/introduction-to-github-copilot/) - Free module on using AI for code generation. `Beginner`

### Tools & Platforms

- [GitHub Copilot Documentation](https://docs.github.com/en/copilot) - Learning AI-powered coding assistance (free for students/open source). `All Levels`
- [Claude Code (Anthropic)](https://docs.anthropic.com/en/docs/claude-code) - AI coding assistant documentation and guides. `All Levels`
- [Cursor Documentation](https://docs.cursor.com/) - AI-first code editor with comprehensive guides. `Beginner`
- [Codeium](https://codeium.com/) - Free AI code completion for individuals. `Beginner`
- [Continue (Open Source)](https://continue.dev/docs) - Open-source AI code assistant for VS Code and JetBrains. `Beginner`
- [Aider Documentation](https://aider.chat/) - Open-source AI pair programming in the terminal. `Intermediate`
