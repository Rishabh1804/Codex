# Graph Neural Networks (GNNs)

> Graph-structured data, network analysis, and learning on relational data.

### Courses

- [Stanford CS224W: Machine Learning with Graphs](http://web.stanford.edu/class/cs224w/) - The comprehensive course on graph ML from Stanford with videos and assignments. `Intermediate`
- [Petar Velickovic: Theoretical Foundations of GNNs](https://www.youtube.com/watch?v=uF53xsT7mjc) - Deep dive into GNN theory by a leading Cambridge/DeepMind researcher. `Advanced`
- [DeepFindr: GNN Tutorials (YouTube)](https://www.youtube.com/playlist?list=PLV8yxwGOxvvoNkzNECKl5YMBFfM4CRGQW) - Practical GNN video tutorials with code walkthroughs. `Beginner`

### Books & Reading

- [Graph Representation Learning Book (Hamilton)](https://www.cs.mcgill.ca/~wlh/grl_book/) - Free comprehensive textbook on graph neural networks. `Intermediate`
- [Geometric Deep Learning: Grids, Groups, Graphs (Bronstein et al.)](https://geometricdeeplearning.com/) - Free book and course on the geometric foundations of deep learning. `Advanced`
- [A Gentle Introduction to GNNs (Distill)](https://distill.pub/2021/gnn-intro/) - Beautiful, interactive explanation of how GNNs work. `Beginner`

### Libraries & Practice

- [PyTorch Geometric (PyG)](https://pytorch-geometric.readthedocs.io/en/latest/) - Leading GNN library with extensive tutorials and examples. `Intermediate`
- [Deep Graph Library (DGL)](https://docs.dgl.ai/tutorials/blitz/index.html) - Flexible GNN framework with great getting-started tutorials. `Intermediate`
- [NetworkX Documentation](https://networkx.org/documentation/stable/tutorial.html) - Python library for graph analysis — good foundation before GNNs. `Beginner`
- [OGB: Open Graph Benchmark](https://ogb.stanford.edu/) - Standardized benchmarks for evaluating graph ML methods. `Intermediate`
- [GraphGym](https://github.com/snap-stanford/GraphGym) - Platform for designing and evaluating GNN architectures. `Advanced`
