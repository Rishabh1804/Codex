# Deep Learning

> Neural networks, architectures (CNNs, RNNs, Transformers), and techniques powering modern AI breakthroughs.

### Courses

- [MIT: Introduction to Deep Learning (6.S191)](http://introtodeeplearning.com/) - MIT's official intro course with lectures, labs, and projects updated annually. `Beginner`
- [fast.ai: Deep Learning for Coders](https://course.fast.ai/) - Practical, top-down deep learning course — build models from day one. `Beginner`
- [Coursera: Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning) - Andrew Ng's 5-course specialization covering CNNs, RNNs, optimization, and structuring ML projects. `Beginner`
- [Stanford CS231n: Deep Learning for Computer Vision](https://cs231n.github.io/) - World-class course on CNNs with lecture notes, assignments, and videos. `Intermediate`
- [Stanford CS224n: NLP with Deep Learning](https://web.stanford.edu/class/cs224n/) - Deep learning applied to natural language with lecture videos and assignments. `Intermediate`
- [NYU: Deep Learning (Yann LeCun)](https://atcold.github.io/NYU-DLSP21/) - Full course by Yann LeCun and Alfredo Canziani with notebooks and video lectures. `Intermediate`
- [UC Berkeley: Full Stack Deep Learning](https://fullstackdeeplearning.com/) - From training models to deploying them in production. `Intermediate`
- [DeepMind x UCL: Deep Learning Lecture Series](https://www.deepmind.com/learning-resources/deep-learning-lecture-series-2020) - Advanced deep learning topics from DeepMind researchers. `Advanced`

### Books & Reading

- [Dive into Deep Learning (d2l.ai)](https://d2l.ai/) - Interactive deep learning book with code in PyTorch, TensorFlow, and JAX. `Beginner`
- [Neural Networks and Deep Learning (Michael Nielsen)](http://neuralnetworksanddeeplearning.com/) - Free online book explaining neural network concepts with clear intuition. `Beginner`
- [Deep Learning Book (Goodfellow, Bengio, Courville)](https://www.deeplearningbook.org/) - The definitive deep learning textbook, freely available online. `Advanced`
- [Distill.pub](https://distill.pub/) - Clear, interactive, beautifully visualized articles on deep learning research. `All Levels`

### Practice & Tools

- [PyTorch: Official Tutorials](https://pytorch.org/tutorials/) - Step-by-step tutorials from basics to advanced model building. `All Levels`
- [TensorFlow Playground](https://playground.tensorflow.org/) - Interactive browser visualization of neural networks — great for building intuition. `Beginner`
- [Papers With Code: Methods](https://paperswithcode.com/methods) - Browse and understand deep learning architectures with linked implementations. `Intermediate`
