# Python for AI

> Python programming foundations essential for machine learning, data science, and AI development.

### Python Fundamentals

- [Python.org: Official Tutorial](https://docs.python.org/3/tutorial/) - The official Python tutorial — comprehensive and always up to date. `Beginner`
- [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/) - Free online book that makes Python practical and fun. `Beginner`
- [Microsoft: Python for Beginners](https://learn.microsoft.com/en-us/training/paths/beginner-python/) - Free learning path from Microsoft. `Beginner`
- [Google: Python Class](https://developers.google.com/edu/python) - Free Python course from Google with exercises. `Beginner`
- [Real Python Tutorials](https://realpython.com/) - High-quality Python tutorials and guides (many free). `All Levels`
- [Codecademy: Learn Python 3](https://www.codecademy.com/learn/learn-python-3) - Interactive Python course in the browser. `Beginner`

### Python for Data & ML

- [NumPy: Official Tutorials](https://numpy.org/learn/) - Learn the fundamental numerical computing library. `Beginner`
- [Pandas: Getting Started](https://pandas.pydata.org/docs/getting_started/index.html) - Official tutorials for the data manipulation library. `Beginner`
- [Matplotlib Tutorials](https://matplotlib.org/stable/tutorials/index.html) - Learn data visualization with Python's core plotting library. `Beginner`
- [Scipy Lecture Notes](https://scipy-lectures.org/) - Free tutorials on the scientific Python ecosystem. `Intermediate`
- [Python Data Science Handbook](https://jakevdp.github.io/PythonDataScienceHandbook/) - Free online book covering the essential data science stack. `Beginner`

### Practice & Development

- [LeetCode: Python Track](https://leetcode.com/problemset/) - Practice coding problems in Python. `All Levels`
- [Kaggle: Python Course](https://www.kaggle.com/learn/python) - Free in-browser Python course with exercises. `Beginner`
- [Project Euler](https://projecteuler.net/) - Math and programming challenges to sharpen your Python. `All Levels`
- [Think Python (Allen Downey)](https://greenteapress.com/thinkpython2/) - Free book on thinking computationally with Python. `Beginner`
