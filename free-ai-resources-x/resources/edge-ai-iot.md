# Edge AI & IoT

> TinyML, on-device machine learning, embedded systems, and deploying AI at the edge.

### Courses

- [Harvard CS249r: TinyML](https://www.edx.org/learn/computer-science/harvard-university-cs249r-tiny-machine-learning-tinyml) - Comprehensive course on ML for embedded systems. `Intermediate`
- [Edge Impulse: Intro to Embedded ML](https://www.edx.org/learn/embedded-systems/edge-impulse-introduction-to-embedded-machine-learning) - Free course on deploying ML to edge devices. `Beginner`
- [NVIDIA Jetson AI Certification](https://developer.nvidia.com/embedded/learn/jetson-ai-certification-programs) - Free AI certification program for edge computing. `Intermediate`
- [Coursera: Introduction to Embedded ML](https://www.coursera.org/learn/introduction-to-embedded-machine-learning) - Free to audit course on embedded ML fundamentals. `Beginner`

### Frameworks & Tools

- [TensorFlow Lite](https://www.tensorflow.org/lite/guide) - Guides for running ML models on mobile and embedded devices. `Intermediate`
- [ONNX Runtime Mobile](https://onnxruntime.ai/docs/tutorials/mobile/) - Cross-platform ML inference for mobile and edge. `Intermediate`
- [Arduino Machine Learning](https://docs.arduino.cc/tutorials/nano-33-ble-sense/get-started-with-machine-learning/) - Tutorials on ML for Arduino microcontrollers. `Beginner`
- [Google Coral: Edge ML](https://coral.ai/docs/) - Documentation and examples for Edge TPU development. `Intermediate`
- [Edge Impulse Studio](https://docs.edgeimpulse.com/) - End-to-end platform for building embedded ML solutions. `Beginner`
- [microTVM (Apache TVM)](https://tvm.apache.org/docs/topic/microtvm/index.html) - ML compiler framework for microcontrollers. `Advanced`

### Reading & Research

- [TinyML Book Resources](https://tinymlbook.com/) - Companion resources for the TinyML book by Pete Warden. `Beginner`
- [MLPerf Tiny Benchmark](https://mlcommons.org/benchmarks/inference-tiny/) - Industry benchmarks for TinyML performance. `Intermediate`
- [Awesome Edge Machine Learning (GitHub)](https://github.com/bisonai/awesome-edge-machine-learning) - Curated papers and resources on edge ML. `All Levels`
- [ARM AI/ML Resources](https://developer.arm.com/solutions/machine-learning-on-arm) - ML optimization for ARM-based devices. `Intermediate`
